🏃‍♂️ Gemini Trail Coach - Performance Cockpit

Bienvenue dans le Gemini Trail Coach, une application de suivi de performance et de coaching intelligent dédiée au Trail et à l'Ultra-Trail. Propulsée par Google Gemini 2.0 Flash, cette plateforme centralise vos données d'entraînement pour vous offrir une analyse experte en temps réel.

🚀 Fonctionnalités Clés

📊 Dashboard de Forme : Visualisation de votre charge d'entraînement (CTL, ATL, TSB) synchronisée via Intervals.icu.

🤖 Coach IA Interactif : Un coach expert disponible 24/7 pour analyser votre semaine et ajuster votre plan.

📅 Planning Hybride : Affichage combiné de vos séances réalisées et de vos entraînements prévus (Nolio).

🏆 Gestion des Objectifs : Calendrier de courses avec priorisation (Objectifs A, B, C) et calcul de J-X.

👤 Profil Expert : Intégration de vos indices de performance (BeTrail, UTMB Index) pour un coaching sur mesure.

🛠️ Installation & Déploiement

Pré-requis

Python 3.9+

Un compte Intervals.icu (API Key)

Un projet Firebase pour l'authentification et la base de données.

Installation Locale

Clonez le dépôt :

git clone [https://github.com/votre-username/MonCockpitPerformance.git](https://github.com/votre-username/MonCockpitPerformance.git)


Installez les dépendances :

pip install -r requirements.txt


Lancez l'application :

streamlit run cockpit.py


🔐 Configuration des Secrets

Pour un déploiement sur Streamlit Cloud, configurez vos secrets (API Keys, Firebase Config) dans l'onglet Secrets du dashboard Streamlit.
